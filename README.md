# 👇 Arduino Giga Display Shield Touch controller Library

The Arduino Giga Display Touch library manages the touch controller of the Giga Display Shield, capturing up to 5 concurrent touch points. 

It acts as a hardware abstraction layer for touch inputs, allowing you to easily capture gestures or build advanced graphical user interfaces (GUIs) by seamlessly integrating [LVGL (v9.x or higher)](https://lvgl.io/).

📖 For more information about this library please read the documentation [here](./docs/).
